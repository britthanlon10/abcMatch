Welcome to ABC Match where you can learn your alphabet and image recognition! How to play: Try to find a picture card that matches the letter shown. For example: If shown the letter A, select the picture that shows something that starts with the letter A, like an apple. A correct answer will add a point to the score; if incorrect a point will be subtracted from the score. Have fun!
